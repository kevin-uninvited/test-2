Chat Widget Integration Demo

Welcome to Your Website

This is a demo of how the chat widget appears on your existing website. The widget will float in the bottom-right corner and won't interfere with your site's design or functionality.

Integration Methods

Method 1: Script Tag

Add this script to your website's HTML:

<script> window.ChatWidgetConfig = { baseUrl: 'https://your-domain.com', brandColor: '#FF6B6B', welcomeMessage: 'Welcome to our store!' }; </script> <script src="https://your-domain.com/embed.js"></script>

Method 2: Direct Iframe

Embed the widget directly with an iframe:

<iframe src="https://your-domain.com/widget?brandColor=%23FF6B6B" style="position: fixed; top: 0; left: 0; width: 100%; height: 100%; border: none; z-index: 9999; pointer-events: none; background: transparent;" ></iframe>

Customization Options

You can customize the widget's appearance and behavior using URL parameters or the configuration object:

// Configuration options { brandColor: '#FF6B6B', // Hex color for widget theme logo: 'https://your-site.com/logo.png', // Your company logo welcomeMessage: 'How can we help?', // Custom welcome message position: 'bottom-right', // 'bottom-right' or 'bottom-left' showOnScroll: true, // Show pulse when user scrolls showOnInactivity: true, // Show pulse after inactivity inactivityDelay: 30000 // Delay in milliseconds (30 seconds) }

Your Website Content

This is where your normal website content would go. The chat widget will overlay on top of your content without interfering with the layout or functionality.

Try scrolling down or waiting 30 seconds to see the widget's smart behavior in action. It will pulse to draw attention when appropriate.

Features

  • Real-time messaging - Instant communication with streaming responses
  • Responsive design - Works perfectly on desktop, tablet, and mobile
  • Smart triggers - Pulses on scroll or inactivity to engage users
  • Customizable branding - Match your brand colors and style
  • Easy integration - No complex setup required
  • Cross-platform - Works with any website or platform
